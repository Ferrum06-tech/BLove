# С Днём Рождения, Яна 🌸

Личный сайт-подарок: главная страница с эффектами, фото-альбом в окошке, музыка и пожелание.

## Структура
```
index.html      — вся разметка (hero / альбом / пожелание)
style.css       — все стили и анимации
script.js       — логика альбома, лепестков и музыки
assets/         — 8 фотографий (photo1.jpg … photo8.jpg)
```

## Как опубликовать на GitHub Pages (бесплатно, 5 минут)

1. Создай новый репозиторий на github.com, например `yana-bday` (можно Private — Pages всё равно работает).
2. Залей туда все файлы из этой папки (через "Add file → Upload files" или `git push`).
3. Зайди в Settings → Pages.
4. В разделе "Build and deployment" выбери Source: **Deploy from a branch**, Branch: **main**, папка **/ (root)** → Save.
5. Подожди 1–2 минуты, обнови страницу — появится ссылка вида:
   `https://твой-логин.github.io/yana-bday/`
6. Открой ссылку, убедись что всё работает (фото, музыка, пожелание).
7. Сгенерируй QR-код на эту ссылку на любом сервисе (например qr-code-generator.com) — готово!

## Как поменять что-то самому

- **Текст пожелания** — переменная `WISH_TEXT` в начале `script.js`.
- **Музыка** — переменная `YT_VIDEO_ID` в `script.js` (это часть ссылки youtube.com/watch?v=ВОТ_ЭТО).
- **Фото** — просто замени файлы в `assets/`, либо поменяй список `PHOTOS` в `script.js`, если меняешь имена/порядок/количество.
- **Подписи под фото** — поле `caption` у каждого фото в массиве `PHOTOS` (можно оставить пустым "").
- **Имя и заголовок** — внутри `index.html` в секции `<section class="hero">`.

## Важно про музыку
Большинство браузеров блокируют автозапуск звука без участия пользователя.
Поэтому видео стартует сразу, но без звука, а звук включается автоматически
при первом касании/клике в любом месте страницы — это самый надёжный способ,
который работает почти всегда. Если по какой-то причине звук не включился сам,
в правом нижнем углу есть кнопка-таблетка "включить звук".
